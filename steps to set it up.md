We’re going to make Windows do the heavy lifting so you don't have to remember to turn this on.

Step 3:
Now we tell Windows exactly what to do:

Go to Actions → New.

Program/script: Paste the path to your python.exe.

Add arguments: Paste the full path to your eye.py file.

Start in (Optional): You can put the folder path here if you want a fancy touch.

Step 4:
Go to the Conditions tab and uncheck "Start the task only if the computer is on AC power." Your eyes deserve protection even when you're unplugged and living on the edge!

Hit OK, and you’re all set! Now, every time you start your PC, it will be there to make sure you keep that 1000000/100 vision.
